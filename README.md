# BS Editor

Ferramenta de edição dos artigos do Meu BS

  - Edição de artigos
  - Criação de novos artigos
  - Gestão das imagens utilizadas


### Tecnologias

BS Editor utiliza tecnologias open source:

* [React] - Library para criação de componentes!
* [node.js] - Back-end
* [Express] - fast node.js network app framework
* [Webpack] - Bundle system

### Instalação

BS Editor requer [Node.js](https://nodejs.org/) v8+ to run.

Instale as dependencias e inicie o servidor.

```sh
$ npm install 
$ npm start
```

Para buildar a versão de produção...

```sh
$ npm install
$ npm build
```
